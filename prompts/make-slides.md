# 论文 / 文档转幻灯片

> 适用：把已有的论文、报告、笔记变成可分享的网页版幻灯片。

```text
[论文/文档所在的文件夹路径]

这是我写过的一篇[论文/报告/笔记]。请把它转化成排版精美的幻灯片并发布到 GitHub Pages。

要求：
1. 尽可能多地保留原文的关键内容、概念、图片、引用；
2. 大约 [N] 页，有封面有封底；
3. 风格：[红白黑配色，无衬线字体，瑞士设计 / 你喜欢的其他描述]；
4. 数学公式要正确渲染；
5. 做完先本地截图自查排版（溢出、重叠、公式渲染），再部署；
6. 发布后验证可访问，把链接给我。
```

**要点**：①「保留原文」那句是灵魂，没有它 AI 会重新创作；②风格用 3–5 个关键词即可；③把「自查再部署」写进要求，成品质量高一档。
