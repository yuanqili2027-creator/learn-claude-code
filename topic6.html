Markdown

用纯文本优雅记笔记，也是和 AI 对话的原生语言

Markdown：用纯文本记笔记，也是和 AI 对话的原生语言

你有没有过这种烦恼：在 Word 里调字体、点粗体按钮、改标题样式，鼠标点来点去，半天没写几个字？Markdown 就是来解决这个问题的。它让你双手不离开键盘，用几个简单的符号就能写出带标题、列表、粗体、代码、公式的漂亮文档。学会它，记笔记会快一倍，和 AI 聊天也会顺手一大截。

一、Markdown 是什么，为什么好用

Markdown 是一种「轻量级标记语言」，由 John Gruber 在 2004 年发明。它的核心思想只有一句话：用纯文本 + 几个约定俗成的符号，来表达文档结构。比如一行前面加 # 就是标题，用 ** 把字包起来就是粗体。

它好在四点：

  1. 纯文本：本质就是记事本能打开的 .txt，永远不会因为软件升级而打不开，几十年后还能读。
  2. 易读：就算不渲染，看源码也一目了然——# 标题 你一眼就知道是标题。
  3. 跨平台：GitHub、知乎、语雀、Obsidian、微信公众号编辑器……到处都认它，复制粘贴不走样。
  4. AI 原生：这是重点，后面单独讲。

和 Word 对比一下就更清楚了：

对比项 Word Markdown
文件本质 二进制，专用软件才能开 纯文本，任何编辑器都能开
排版方式 鼠标点按钮、选样式 键盘敲符号
版本管理 难以对比改动 一行行 diff 清清楚楚
专注度 容易陷入调格式 专注写内容

二、完整语法逐项讲

下面每个例子都用「代码块」展示 Markdown 源码，你照着敲即可。

标题（一到六级）

用 # 开头，几个 # 就是几级标题，# 后面要空一格：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

段落与换行（新手第一大坑）

Markdown 里，必须用空行来分段。 你在编辑器里按一次回车，渲染出来往往还是同一段！看这个反例：

第一行
第二行

上面渲染出来会挤成一行「第一行 第二行」。正确写法是中间空一行：

第一行

第二行

如果你确实想「换行但不换段」（同一段内硬换行），在行尾打两个空格再回车：

第一行（这里行尾有两个空格）  
第二行

粗体、斜体、删除线

**粗体** 或 __粗体__
*斜体* 或 _斜体_
***粗斜体***
~~删除线~~

效果：粗体、斜体、粗斜体、删除线。

有序列表 / 无序列表 / 嵌套

有序用 数字.，无序用 -（* + 也行），符号后都要空一格：

1. 第一项
2. 第二项
3. 第三项

- 苹果
- 香蕉
- 橙子

小贴士：有序列表的数字不必写对，全写 1. 也会自动排成 1、2、3。嵌套只要缩进（一般 2 或 4 个空格）：

- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 白菜

任务列表

方括号里空格是未完成，x 是已完成：

- [ ] 还没做的事
- [x] 已经做完的事

行内代码与代码块

单个反引号包住是行内代码，三个反引号包住是代码块，开头写语言名可获得语法高亮：

这是 `行内代码`。

```python
def hello():
    print("Hello, Markdown!")
```

引用、分割线

> 这是一段引用。
> 引用可以有多行。

---

---（也可以是 *** 或 ___）单独一行就是一条水平分割线。

链接与图片

图片就是链接前面加一个 !，方括号里是「替代文字」：

[百度一下](https://www.baidu.com)
![一张图片的说明](https://example.com/photo.png)

表格

用 | 分隔单元格，第二行用 --- 分隔表头，: 可控制对齐：

| 姓名 | 年龄 | 城市 |
| :--- | :--: | ---: |
| 小明 | 18 | 北京 |
| 小红 | 20 | 上海 |

转义字符

想显示符号本身（比如真的要打一个 *），前面加反斜杠 \：

\*这里不会变成斜体\*

三、数学公式：$ 与 $$

Markdown 用 LaTeX 语法写数学公式。行内公式用单个 $ 包裹，跟文字排在一行；块级公式用 $$ 包裹，会单独居中占一行：

质能方程是 $E = mc^2$，写在句子里。

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

注意：公式渲染依赖 KaTeX 或 MathJax 这类引擎，普通记事本不认，需要在支持它的环境（如 Obsidian、语雀、支持数学的 Markdown 预览器）里才显示成漂亮公式。

常见 LaTeX 数学符号简表：

用途 写法 效果
上标 x^2 x 的平方
下标 x_i x 下标 i
分数 \frac{a}{b} a 分之 b
求和 \sum_{i=1}^{n} 从 i=1 求和到 n
根号 \sqrt{x} 根号 x
希腊字母 \alpha \beta \pi α β π
无穷 \infty ∞
乘除 \times \div × ÷

四、为什么说 Markdown 是「AI 的原生语言」

你留意过没有：ChatGPT、Claude 这些 AI 回复你时，标题、列表、粗体、代码块——本身就是用 Markdown 写的，只是聊天界面帮你渲染好了。因为这些模型的训练数据里有海量 Markdown 文本，它们「天生」就习惯用这套格式来组织答案。

所以对你有两个好处：一是看得懂，你把 AI 回复复制到笔记软件里，格式几乎完美保留；二是问得好，你提问时用 Markdown 把需求列成标题和列表，结构清晰，AI 理解得更准、回答得更有条理。可以说，学会 Markdown = 学会了和 AI 高效对话的通用语。

五、常用工具

  • VS Code：程序员最爱，写好 .md 后按 Cmd+K 再按 V 打开侧边实时预览（或点右上角的预览小图标）。
  • Typora：所见即所得，边写边渲染，几乎看不到源码，最适合新手。
  • Obsidian：本地知识库神器，支持双向链接和数学公式，做长期笔记首选。
  • 语雀 / 飞书文档：国内在线协作平台，天然支持 Markdown，适合团队。
